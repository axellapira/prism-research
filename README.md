# Prism Research

Benchmark and paper artifacts from Scrambld, Inc. (Prism).

## What's here

This repository contains the methodology, code, and data for evaluations of
[Prism](https://tryprism.com), an AI-native recruiting agent built by
[Scrambld, Inc.](https://tryprism.com).

The repository is **public from day one** so that pre-registrations,
methodology decisions, and evaluation prompts are visible (and immutable
via git history) before any results are run. This is the credibility
contract: nothing in this repo gets quietly edited after the fact.

## Current work

- **`pre-registration/`** — methodology pre-registrations for upcoming
  evaluations. Every benchmark we publish is preceded by a committed
  pre-reg in this directory. The git timestamp is the authoritative
  "this was decided BEFORE the run" stamp.

## Upcoming

A technical report on Prism's performance on
[PeopleSearchBench](https://arxiv.org/abs/2603.27476) is in preparation.
Pre-registration coming soon.

## Reproducing

Reproduction guides will live alongside each paper, in the relevant
sub-directory. They will list every dependency, every API key, every
prompt, every harness commit hash, and every Prism build SHA used.
